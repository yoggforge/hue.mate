Hue.mate

Pick colors from anywhere in your browser. Save, organize, and copy in a click.

What is Hue.mate?

Hue.mate is a Google Chrome extension that lets you pick any color you see in your browser and instantly get its code. Build your own color palettes, copy codes with one click, and stay in the flow while you design or develop.

Features

Feature Description
Pick colors in-browser Grab colors from anywhere in the browser and get the exact color code.
Copy in one click Click a saved color to copy its code to your clipboard instantly.
Save to palette Store picked colors in the built-in palette for quick access.
Multiple palettes Create up to 3 separate color palettes for different projects.
Clear palette Reset a palette whenever you want to start fresh.
